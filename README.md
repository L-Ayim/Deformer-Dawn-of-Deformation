# Deformer: Dawn of Deformation

Deformer is a WebGL demo that showcases a deformable landscape with online multiplayer.

## Getting Started

1. Install dependencies with `npm install`.
2. Start the web and WebSocket servers with `npm start`.
3. Open <http://localhost:3000> in a modern browser.

## Features

- Sharp textures through anisotropic filtering.
- Mobile-friendly controls and clean UI.
- Procedurally generated terrain using Simplex noise.
- Configurable pixel ratio and antialiasing.
- Boomerang combat with recall and optional terrain hazards.
