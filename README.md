## Setup

1. Install [Node.js](https://nodejs.org/) 18 or newer.
2. Run `npm install` to grab the dependencies.

## Running

The demo uses a small WebSocket server alongside a static site. Start both with:

```bash
npm start
```

This launches the game at <http://localhost:8080/> and the socket server on
port `3000`.

## Gameplay

Move with **W/A/S/D** and look around with the mouse. Hold **Shift** to sprint.
Tapping **Space** twice toggles fly mode. Click (or tap the red button on mobile)
to charge and throw a boomerang that deforms the terrain. Hit the yellow target
pillars to score points.
